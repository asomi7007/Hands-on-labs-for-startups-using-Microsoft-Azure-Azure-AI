# 바로 써먹는 Azure & Azure AI 활용 — Material for MkDocs

이 리포는 Azure & Azure AI 실습 블로그 템플릿입니다.

## 빠른 시작
```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
mkdocs serve
```
- http://127.0.0.1:8000

## GitHub Pages 배포
- main/master 푸시 → Actions → `gh-pages` 브랜치 생성 → Settings → Pages → gh-pages
