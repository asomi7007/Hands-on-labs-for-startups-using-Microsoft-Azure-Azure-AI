# Azure Marketplace에서 WordPress 배포

## 옵션
- Azure App Service 기반의 WordPress
- 가상머신(Compute) 기반의 WordPress (초심자 비추천)

## 절차 (App Service)
1. Azure Portal → Create → **WordPress** (Marketplace)
2. App Service, Database (Flexible Server) 구성
3. 배포 후 `https://<appname>.azurewebsites.net` 접속
4. 초기 관리자 계정 설치/로그인
