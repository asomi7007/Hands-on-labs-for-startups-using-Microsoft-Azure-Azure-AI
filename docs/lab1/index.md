# Lab 1 · Microsoft for Startups Founders Hub & Azure 크레딧 받기

## 목표
- Founders Hub 가입 및 **Azure 크레딧 활성화**
- Azure Portal에서 구독(Subscription) 확인

## 준비물
- GitHub 또는 Microsoft 계정
- 회사/팀 기본 정보

## 단계
1. Founders Hub 페이지 접속 → 계정 로그인
2. 팀/회사 정보 입력, 제출
3. **Benefits**에서 Azure 크레딧 활성화
4. Azure Portal → **Subscriptions**에서 크레딧 적용된 구독 확인

## 체크리스트
- [ ] Benefits 활성화 완료
- [ ] 구독에 크레딧 표시
- [ ] 과금 경보(Alert) 설정
