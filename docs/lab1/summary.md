# Summary

Lab 1을 완료하면 전용 Microsoft 계정으로 Founders Hub 혜택을 신청해 Azure 구독에
$1,000 크레딧을 성공적으로 연결한 상태가 됩니다. 다음 항목을 확인해 온보딩이 제대로
마무리됐는지 점검하세요.

## What we built

- 전용 Outlook, LinkedIn, GitHub(선택) 계정을 생성했습니다.
- Founders Hub `Get started quickly` 흐름을 완료하고, 새 Azure 구독을 발급받았습니다.
- Cost Management에서 잔액을 추적할 수 있도록 `$1,000 Azure 크레딧`을 활성화했습니다.

## Next steps

- Azure CLI 또는 Azure Developer CLI를 설치해 `az login`으로 새 구독을 연결합니다.
- Lab 2 이상에서 사용할 리소스 그룹, AI 서비스 등을 배포할 예정이므로 구독의 지역(Region) 기본값을 미리 확인합니다.
- 팀 구성원이 있다면 동일 가이드에 따라 계정을 준비하도록 공유하고, Trello/Planner 등 협업 도구로 진행 상태를 관리합니다.

## Troubleshooting

- **크레딧 미표시**: 30~60분 대기 후 `비용 관리 + 청구`를 새로고침하고, 그래도 표시되지 않으면 Founders Hub 포털에서 혜택을 재실행합니다.
- **MFA/로그인 오류**: 브라우저 프라이빗 창을 새로 열고 다시 로그인합니다. 필요하다면 `보안 정보`에서 MFA 수단을 재설정합니다.
- **결제 수단 거절**: 다른 카드 또는 체크카드를 사용하고, 여전히 실패하면 은행에 국제 소액 승인 허용 여부를 확인합니다.
- **기술 지원 요청**: 계정 이메일, 오류 메시지, Azure 포털 스크린샷을 포함해 `admin@eldensolution.kr`로 문의합니다.
