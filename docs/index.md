# 바로 써먹는 Azure & Azure AI 활용

이 문서는 스타트업/개발팀이 **바로 적용할 수 있는 Azure & Azure AI 실습**을 다룹니다.
각 Lab 별로 **목표 → 준비물 → 단계별 실습 → 체크리스트 → 트러블슈팅**을 제공합니다.

!!! info "구성"
    - Lab 1: Founders Hub 가입 · Azure 크레딧 받기
    - Lab 2: WordPress 구축 + Azure AI 에이전트/챗봇 통합
    - Lab 3: Contoso Sales Assistant 체험
    - Lab 4: Static Web Apps 배포
    - Lab 5: Node.js 앱 GitHub → Azure 배포
